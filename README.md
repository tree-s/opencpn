# Scoop Bucket Template

[![Tests](https://github.com/tree-s/opencpn/actions/workflows/ci.yml/badge.svg)](https://github.com/tree-s/opencpn/actions/workflows/ci.yml) [![Excavator](https://github.com/tree-s/opencpn/actions/workflows/excavator.yml/badge.svg)](https://github.com/tree-s/opencpn/actions/workflows/excavator.yml)

OpenCPN bucket for [Scoop](https://scoop.sh), the Windows command-line installer.

## How do I install these manifests?

After manifests have been committed and pushed, run the following:

```pwsh
scoop bucket add opencpn https://github.com/tree-s/opencpn
scoop install opencpn/opencpn
```

## How do I contribute new manifests?

To make a new manifest contribution, please read the [Contributing
Guide](https://github.com/ScoopInstaller/.github/blob/main/.github/CONTRIBUTING.md)
and [App Manifests](https://github.com/ScoopInstaller/Scoop/wiki/App-Manifests)
wiki page.
